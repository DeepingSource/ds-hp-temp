## 1. 핵심 판단 — 왜 중요한가
기능소개 v7 표지 문구: **"화면 · 실제 라이브 대시보드에서 캡처 (mock 없음)"**.
현재 saai-agent 제품 페이지(`StoreAgentView.tsx`)와 목업 갤러리는 mockup 컴포넌트 기반. **실제 화면 > 목업**은 B2B 신뢰도에서 결정적 차이 → 신규 캡처로 교체 시 제품 페이지 설득력이 크게 오른다. 리드 퍼널 ② 신뢰(MOFU) 단계 직접 강화.

## 2. 반영 맵핑 (신규 자료 → 사이트 위치)

### M1. saai-agent 제품 페이지 목업 → 실제 캡처 교체 [핵심]
- **소스**: 기능소개 v7의 30여 캡처. 대표 장면 선별(중복 제거 후 8~12장):

  - 발주 추천 패널(#15) · 폐기 분석(#16) · 선반 재배치 ROI + Before/After GIF(#20)
  - 대화형 직접배치 드래그&드롭(#21) · 다중매장 팬아웃(#13) · 일본어 UI(#10) · 냉동고 고장 예측(#31, 차별기능)
- **작업**: `StoreAgentView.tsx` 목업 섹션을 실제 스크린샷 갤러리/캐러셀로 교체. 이미지는 `public/images/`에 최적화 저장(webp), alt 텍스트 3로케일.
