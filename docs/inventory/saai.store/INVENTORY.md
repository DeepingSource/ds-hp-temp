- **렌더 스택 혼합** — 마케팅(`/landing`)·콘텐츠(`/archive`) 는 Vite 기반 React SPA + 빌드타임 prerender, 제품 앱(`/pop`)은 Next.js. 동일 도메인에서 두 스택을 라우팅 분리 운영.
- **공통 자산** — `/saai-logotype.png`, `/saai-symbol.png`, `/favicon.svg?v=2`, `/og-image.jpg?v=2`.
- **외부 임베드** — 모든 페이지에 `https://mcp.figma.com/mcp/html-to-design/capture.js` 가 비동기 로드 (Figma "HTML to Design" 캡처 헬퍼 — 디자인 작업용으로 임시로 들어있을 가능성, 운영 시 제거 권장).

- **SEO 검증** — `naver-site-verification` 메타 토큰 부착 (`0a49f4d00c65944cc234bee478bfe784d0bac32f`).
- **소셜 OG** — Twitter `summary_large_image` 카드와 `og:locale=ko_KR` 일관 적용.

## 3. `/landing` 과 `/pop` 메타데이터 분석 (SPA 본문 추정)

SPA 셸이라 본문 카피는 추출되지 않았지만, 메타 태그·내비·아카이브 카피·뉴스레터 폼의 흔적으로 페이지 의도를 역추론.

### 3.1 `/landing/` — 제품 마케팅 홈

| 필드 | 값 |
|---|---|
| `<title>` | `SAAI — 편의점 AI 비서 \| 발주 추천, POP/홍보물, 트렌드` |
| `meta description` | `편의점 발주가 막막할 때, POP·홍보물 만들기 어려울 때, 신상품이 궁금할 때 — 모든 운영 노하우를 담은 편의점 AI 비서, SAAI` |
| `og:image` | `https://saai.store/og-image.jpg` (1200×600, alt: "SAAI — 편의점 AI 비서") |
| `og:locale` | `ko_KR` |

**역추론한 제품 3대 기능** (제목·메타·내비에서 일관):
1. **발주 추천** — "편의점 발주가 막막할 때" → AI 발주 추천 엔진
2. **POP/홍보물 제작** — "POP·홍보물 만들기 어려울 때" → POP 메이커(=`/pop`) 진입점
3. **신상품/트렌드** — "신상품이 궁금할 때" → 스토어레터·트렌드 Fit! 진입점

| 필드 | 값 |
|---|---|
| `<title>` | `SAAI — 편의점 AI POP 메이커 \| 무료 POP, 홍보물 제작` |
| `meta description` | `편의점 POP 만들기 어려울 때, 편의점 홍보물 제작 필요할 때 — POP를 무료로 디자인해드리는 편의점 AI 비서, SAAI` |
