| 모듈 (URL) | 한국어 명 | 핵심 기능 | 차별점 | 주요 타깃 사용처 | 홈에서의 위치 |
|---|---|---|---|---|---|

**해석**: 4개 모듈은 **단일 SaaS의 기능 단위**이며, 홈에서 사용자가 "내 매장에 필요한 기능"을 카드로 클릭해 모듈 페이지로 진입하면 → 알림 UI 데모를 보고 → 다시 홈 하단 요금제로 돌아와 결제하는 전형적인 SaaS 마케팅 펀넬을 형성합니다. 모듈 페이지에는 **개별 가격이 없고** 가격은 항상 홈에 집중되어 있습니다.

**옵션 A — 외부 분기 유지 (권장)**
