# /company/about 섹션별 병합·확정 스펙 v1

작업일: 2026-07-21 · 대상: `/ko/company/about`
목적: **기존 콘텐츠(`about.yaml` + 컴포넌트) + 연결 그림(`about_saai_bridge.html`)** 을 실제 페이지로 **합치는 최종 스펙.** 계획서 v3의 두괄식 7장 골격을 카피·링크·자산까지 확정.
진행 방식: **①부터 한 장씩 확정.** 확정된 장은 ✅, 확정 대기는 ⏳.
읽는 법: 각 장에 **[최종 카피] · [기존→변경] · [병합/자산] · [링크·디자인]** 4블록.

전체 골격: ① 선언(다크) → ② 비전(강조) → ③ 향후 5년(다크·그림상단) → ④ 왜냐하면 → ⑤ 누가 → ⑥ SAAI(그림하단) → ⑦ Contact(다크)

## ① 선언 — 우리는 오프라인을 다시 만든다  ✅ 확정

### [최종 카피]
- (eyebrow) **About DEEPINGSOURCE**
- (H1·대형) **REINVENT OFFLINE**
- (국문 리드 · 2줄 스택, "다시" 파랑 강조) — **옵션 A 확정**
 ```
 오프라인을
 다시 만듭니다        ← '다시'만 primary(#376AE2)
 ```
- (대비 카드 2장 — 유지)
 - 왼쪽: `온라인은` → **측정으로 최적화됐다**
 - 오른쪽: `현장은` → **거의 그대로였다**
- (클로징 1줄) **딥핑소스는, 그 현장을 다시 만듭니다.** — 딥핑소스 살림(확정)

### [기존→변경]
| 키(about.yaml) | 기존 | 변경 |
|---|---|---|
| `missionStatementSub` | 오프라인을 다시 만든다 | **우리는 오프라인을, 다시 만듭니다** (선언형으로) |
| 대비 카드 4키 | 온라인은/측정으로 최적화됐다 · 현장은/거의 그대로였다 | **유지** (온라인↔현장 대비를 말하는 유일한 자리) |
| `companyIntro` | "온라인은 지난 20년…현장은 거의 그대로였습니다" | **삭제** — 대비 카드와 같은 말 반복(과적의 주범) |
| `companyIntro2` | 딥핑소스는 그 현장을 다시 만듭니다 | **유지 → 클로징 1줄로** (약간 다듬음) |
| `companyLine`(brand-canon) | 2018년부터 영상의 익명화를 깎아 온 회사 | ①에서 **빼서 ④(왜냐하면)로 이관** |
| `perfectSpace` 화살표 | 당신의 공간 → 모든 공간 | ①에서 **빼서 ②(비전)로 이관** |

### [병합/자산]
- 연결 그림은 ①에 **없음**(그림은 ③·⑥). ①은 순수 히어로 정리.
- `MasterStatement`/히어로 컴포넌트에서 하단 3연속 문구(companyLine·companyIntro·perfectSpace) 블록 제거 → 카드 아래 **클로징 1줄만**.

### [링크·디자인]
- 아웃바운드 링크 없음(선언은 잡아두는 자리).
- 배경 **다크**(`surface-dark`). H1 = 타이포 **L1**(`text-5xl~7xl`).
- 요소 6개 → **4개**(eyebrow·H1·리드·카드·클로징). 카드 아래 여백 크게 = "숨".
- 대비 카드: 왼쪽 중립(white/5), 오른쪽 **primary 톤**(현행 유지) — "현장"에 파랑 앵커.

### [확정 로그 ①]
- 국문 리드 = **옵션 A**(2줄 스택, '다시' 파랑). 쉼표 제거·줄바꿈 처리.
- 클로징 = "딥핑소스는, 그 현장을 다시 만듭니다" (딥핑소스 유지).
- `companyIntro` **삭제 확정**. `companyLine`→④ · `perfectSpace` 화살표→②.

## ② 우리가 가려는 곳 — 모든 공간을, 완벽하게  ✅ 확정

### [최종 카피]
- (eyebrow) **Vision** — Mission은 ① 최상단 선언으로 분리(확정)
- (H2·대형) **모든 공간을, 완벽하게.** — About 안에서 이 문장은 **여기 1회만**
- (전개 1줄, 텍스트로 풀기) **당신의 공간에서 시작해, 모든 공간으로.** — → 기호 대신 문장
 - ↑ `perfectSpace`(your→every) — ①에서 이관. ⑥ SAAI 브릿지의 복선(같은 your→every).
- (믿음 본문) **측정할 수 있으면, 나아질 수 있습니다. 온라인을 바꾼 그 힘을, 이제 물리 세계로.**

### [기존→변경]
| 키(about.yaml) | 기존 | 변경 |
|---|---|---|
| `vmEyebrow` | Vision & Mission | **Vision** (Mission은 ①로) |
| H2(현행 `perfectSpace.every`) | 모든 공간을, 완벽하게 | **유지** — 단 About 유일 등장 |
| `vision` | 당신의 공간에서, 모든 공간으로 | **유지 → 화살표 전개**(your→every) |
| `mission` | 측정할 수 있으면… 물리 세계로 | **유지 → 믿음 본문으로** (③ closing과 중복 금지 근거) |
| `missionStatement`/`Sub` | REINVENT OFFLINE / 오프라인을 다시 만든다 | ②에서 **제거**(①과 중복) |
| VisionDiagram | (V/M 섹션 중앙) | **③으로 이동**(다크에서 제 옷) |

### [병합/자산]
- VisionDiagram 빠짐 → ②는 **텍스트 중심**으로 가볍게. 시각물은 "당신의 공간 → 모든 공간" **작은 2단 그래픽**(브릿지 복선) 정도만.
- 연결 그림 없음(그림은 ③·⑥).

### [링크·디자인]
- 아웃바운드 링크 없음.
- 배경 **밝음-강조**. 이 섹션이 §5-3 **파랑 앵커 자리** — "모든 공간을, 완벽하게" 블록 하나를 **파랑 채움/좌측 바**로 못박음.
- 타이포 **L1~L2**(비전은 크게). ①(다크)→②(밝음) 전환 = 첫 명암 대비.

### [확정 로그 ②]
- eyebrow = **Vision** 단독. Mission은 ① 최상단.
- 전개 = **텍스트로 풀기** "당신의 공간에서 시작해, 모든 공간으로."
- 믿음 본문 ②에 확정 → ③ closing은 "권고는 AI가, 결정은 사람이"로 분리.
