## 0. 전체를 관통하는 렌즈 (이번 세션)

- **독자·목표:** 회사명 검색·AEO/SEO로 처음 온 방문객 → **전환**. 전환 1순위 = **프랜차이즈·체인 본사**.
- **핵심 포지셔닝(그라운딩):** 범용 언어 AI(GPT·Claude)는 *당신 공간의 신호*를 못 받아 원론적 답을 한다. **SAAI는 공간을 그대로 읽고 이해해 '당신의 답'을 하고, 실행까지 한다.** Physical AI 뉘앙스, 단어는 미사용.
- **전환 동선:** 모든 길 → **/enterprise(본사 맞춤 제안·문의)** 또는 **/contact**로 수렴.

## 1. 블록별 확정 요약

### A. 랜딩 (`랜딩_전환재정렬_v2.md`) — ✅ 스펙 완료
- **히어로:** 질문형 H1 **"지금 쓰는 AI는, 당신의 〔매장·현장·전시장·물류센터·카페·무인매장〕을 알고 있나요?"**(회전 호명) + 신호→당신의 답 서브 + **S·A·A·I 선형**(Spatial Anonymized Agentic Intelligence). 고정 H1(크롤링용)="당신의 공간을". 시네마틱 다크 부분배경.
- **오리엔테이션 밴드 신설**(그라운딩 설명 + AEO 4팩트 + 본사 앵커 + 경쟁 AI 실명 각주).
- **문제제기** 본사 스케일 한 겹 · **클로징** REINVENT OFFLINE kicker. KO/EN/JP 완료.

### B. 제품·엔터프라이즈 — ✅ 스펙 완료
- **모듈 체계**(`제품_모듈체계_네이밍_v2.md`): `saai suite`(코어, 전부 다) + **모듈(Modules)** = saai 제품(insight·care·agent·ads insight…) + store 기능(count·queue·pop·fit…). **종속 없음**(필요한 것부터, count 입문 회복). `saai tool` 폐기. "필요한 것부터, 전부까지 — 계속 늘어납니다." saai ads insight = 제품 모듈(번역 완료·보강만).
- **/enterprise 재설계**(`enterprise_재설계_v1.md`): 스파인 **"당신의 체인에 맞는 제안"** + 10섹션 IA. CTA = **제안 요청(폼)+이메일**(예약 위젯 보류). 리드 파이프라인 = **Slack(구현됨·설정만) + CSV/시트(신규) + HubSpot(신규)**, `source` 태깅.

### C. 기술/모델 (`technology_models_콘텐츠_v1.md`) — ✅ 스펙 완료

### D. 회사 서사 (`회사서사_about_team_news_v1.md`) — ✅ 스펙 완료
