- 문제 제기 4종을 care 톤(SMB 친밀)으로: "고객이 떠난 뒤에야 알게 되는 진열" / "퇴근 후엔 온도를 확인할 방법이 없어요" / "녹화만 되고, 들여다볼 시간은 없죠" / "직원마다 기준이 달라요"
- "30분→8분" 수치는 **예시로 사용 가능** (확정): "자사 실험 매장 기준 예시" 단서 필수. 예: "하루 점검 30분이 8분으로 — 자사 실험 매장 기준 예시입니다."

**3d. store-count View (inline dict — 코드)**:
- 핵심 프레임: "카운터에서 의사결정 보조자로" · **유입률 = 입장객 ÷ 통행 인구** ("통행 인구는 입지의 기초 체력, 입장객은 매장의 실제 수요") · "5분 무공사 설치 · 영상은 저장하지 않고 통계만"
- 정확도 95~97%는 검증 3건(편의점·전시장·복합공간) 출처 표기 가능할 때만 (★규칙).
- 법 근거(제25조, 동의 불요)는 technology 또는 FAQ로.

**3e. `saai.yaml`(B2C)·`products.yaml`(허브)**: 현행 구조 유지, WP0 네이밍 표기만 반영. /products 허브의 SAAI 브랜드 스토리 재편은 SITE_IMPROVEMENT_PLAN P2-1 트랙과 중복 — 그쪽에 위임.

### WP4. 업종 4종 (`retail` · `drug` · `food-beverage` · `large-space.yaml`) — 수치 반영 허용 구역
자료 사례 수치 + "예시" 디스클레이머 (모두 "고객사 사례 기준, 매장 조건에 따라 다를 수 있습니다" 각주):
- retail(편의점): 전환율 높은 진열대를 통행 많은 곳으로 — 매출 +20% · retail(패션/스포츠): 픽업 데이터 기반 결제 시 추가 제안 — 객단가 +10%
- drug: 첫 검토 히트맵(20대 남성=향수, 여성=스킨) 기반 HOT ZONE 배치 — 매출 +15%
- food-beverage: 4인 테이블 평균 1.1~1.8명 → 2인 테이블 전환 + 1·2인 세트 — 이용객·매출 +10%
- large-space(전시): 테마·팝업 관심도 정량화 — 방문객 +30%
※ %값 원본 확인 완료 (2026-07-16 확정): +20% / +15% / +10% / +10% / +30% 그대로 반영.

### WP5. 사례 — Golden Case 확장 (`content/case-studies/*.mdx`)
현행: 5케이스×3언어(cvs-100, smb-53, drug-translate, cafe-sync, space-remeasure), goldenStep은 "고정 슬롯 아닌 분류 태그" — **mdx 추가만으로 카드 자동 확장**.
- 매핑: 자료 사례 ↔ 기존 케이스 — 편의점 +20%↔cvs-100(보강), 드럭 +15%↔drug-translate(보강), 전시 +30%↔space-remeasure(보강), F&B +10%↔cafe-sync(보강 또는 신규 fastfood 케이스), **패션/스포츠 객단가 +10% = 신규 mdx** (goldenStep: translate 또는 sync 제안).
- 각 케이스 구조: 업종/문제/처방/결과 + `metrics[{label,value,verified}]`. 자료 수치는 **verified: false + "예시" 단서**로 시작, 고객 검증 후 true 승격.
- **정합 수정 (확정)**: `smb-53-ko/en/jp.mdx`의 "30분→8분" metric을 `verified: true → false`로 강등하고 "자사 실험 매장 기준 예시" 단서 병기 (소개서 단서와 정합).
- StoreAgent v7의 실측 소재(올리브영 성수 vs 홍대 벤치마킹, 냉동고 이상 예측 등)는 "점주의 하루" 콘텐츠(§5-2)로.

### WP6. 요금 (`pricing-data.ts` + `pricing.yaml`) — 신규 추가
- store count 요금이 현재 코드에 없음 → **B2B 영역에 신설**: "store count — 월 30만 원부터(Standard). 매장 수·계약 기간에 따라 조정되니, 상담으로 맞는 구성을 잡아드립니다." CTA "요금 상담 신청". Pro(40만, LTE 실시간)·Connect(기존 CCTV 연동, 견적)는 부가 옵션 수준으로만.
