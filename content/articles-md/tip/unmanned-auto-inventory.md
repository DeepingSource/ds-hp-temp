## 무인매장 재고 관리의 핵심 과제

무인매장은 상주 직원이 없기 때문에 진열대 결품을 즉시 인지하기 어렵습니다. 고객이 상품을 집어 들었지만 빈 자리가 채워지지 않으면, 다음 고객은 그대로 이탈합니다. 실제 무인 편의점 운영 데이터에 따르면 결품 1시간당 평균 3.2건의 구매 기회가 사라집니다.

- **무인매장 평균 결품률**: 8.7% _(유인매장 3.8%)_
  - **결품 시 고객 이탈률**: 42% _(대체 구매 28%)_
  - **결품 인지 평균 시간**: 4.2시간 _(수동 점검 기준)_

---

## CCTV 기반 결품 감지 원리

AI가 진열대 영상을 분석하여 상품 수량 변화를 실시간으로 추적합니다. 특정 칸의 상품이 설정 수량 이하로 줄어들면 즉시 알림이 발생합니다. 카메라 1대로 최대 3개 진열대를 커버할 수 있으며, 야간 저조도 환경에서도 95% 이상의 감지 정확도를 유지합니다.

> 💡 **감지 정확도를 높이는 팁**
>
> 진열대 조명을 300lux 이상으로 유지하고, 카메라 각도를 진열면 기준 30~45도로 설정하면 상품 구분 정확도가 크게 향상됩니다.

## 자동 발주 프로세스 설계

결품 감지 후 자동 발주까지 연결하려면 3단계 워크플로우가 필요합니다.

- [ ] 1단계: CCTV 감지 → 결품 품목 자동 식별 (SKU 매칭)
- [ ] 2단계: 재고 기준선 대비 부족량 계산 → 발주 수량 산출
- [ ] 3단계: 점주 승인 또는 자동 발주 실행 → 입고 일정 알림

자동 발주를 도입한 무인매장의 경우 결품 지속 시간이 평균 4.2시간에서 38분으로 단축되었습니다. 불필요한 과잉 재고도 15% 감소하여 폐기 비용까지 절감됩니다.

## 운영 효율화 체크리스트

- [ ] 주요 매출 품목 상위 20개에 우선 감지 설정
- [ ] 시간대별 판매 속도 데이터로 발주 기준선 자동 조정
- [ ] 주 1회 감지 정확도 리포트 확인 및 카메라 각도 미세 조정
- [ ] 입고 후 진열 완료까지 알림 연동
