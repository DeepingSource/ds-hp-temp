After logging in to store agent, select the store you want to analyze from the top bar. If you run several stores, the store switcher in the top bar lets you move quickly to another store. Chat answers and overview metrics update together based on the store you select.

## Key features

- **Store selection dropdown** — click the store name on the left of the top bar to select your store.
- **Store switching** — when you run several stores, switch straight to another one from the dropdown.
- **User info display** — the account you are logged in with and its permissions are shown on the right of the top bar.
- **Log out** — log out safely using the button at the far right of the top bar.

## How to use it

1. Log in with your account information.
2. Open the store name dropdown on the left of the top bar.
3. Select the store you want to check from the list, and the whole screen updates to that store.
4. When you are done, click the log out button at the far right.

## Tips

- When you switch stores, both the chat answers and the overview metrics change to the selected store.
- If you handle several stores, make sure you have the right store before you start your analysis.
