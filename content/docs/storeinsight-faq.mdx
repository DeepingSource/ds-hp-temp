## 로그인이 안될 경우

- 사용자 ID와 비밀번호가 정확한지 다시 확인하세요.
- 키보드의 Caps Lock 키가 꺼져 있는지 확인하세요.
- 사용 중인 웹 브라우저의 캐시를 삭제한 후 다시 시도해 보세요.

## 데이터가 표시되지 않을 경우

- 리포트의 조회 기간이 올바르게 설정되어 있는지 확인하세요.
- 적용된 필터를 초기화한 후 다시 시도해 보세요.
- 페이지를 새로고침(F5) 해보세요.

## 성능 관련 문제

- 웹 브라우저를 최신 버전으로 업데이트하세요.
- 불필요하게 열려있는 다른 브라우저 탭을 닫아 주세요.
- 인터넷 연결 상태가 안정적인지 확인하세요.

---

해결되지 않는 문제는 **시스템 관리자** 또는 **딥핑소스 고객지원**으로 문의해 주세요.
