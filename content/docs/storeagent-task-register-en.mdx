A Store Agent answer isn't just something to read and move on from — you can turn it into an actionable task. When a recommendation is worth acting on, use "Save as Task" to add it to your to-do list, then attach an owner and a schedule to manage it.

## Key features

- **Save as Task button** — The button below an AI answer moves the answer's content into a new task.
- **Task detail entry** — Register a task by specifying the store, title, timing, and owner.
- **Adopt a recommended action** — Pick one of the action items the AI suggested and use it directly as a task.

## How to use

1. Find the AI answer you want to act on.
2. Tap "Save as Task" below the answer.
3. In the new task window, enter a title, a timing, and an owner.
4. Tap "Register" to add it to your task list.

## Tips

Write the title with a specific what and when so it's easy to recognize later. If you only want to keep something for reference, register it as a memo instead of a task.
