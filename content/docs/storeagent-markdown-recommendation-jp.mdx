賞味期限が迫っている、またはシーズンが終わりに近づいている商品は、そのままにしておくと廃棄につながります。見切り推奨は、割引販売に回す商品と発注量の調整が適した商品を区分して提案します。予想廃棄損失を根拠に、今何から処理すべきかを整理します。

## 主な機能

- **見切り対象の選別** — 廃棄率と直近の売上の流れを基準に、すぐに割引する商品を選び出します。
- **割引 vs 回転調整** — 割引が適した商品と、発注量の削減が適した商品を区分します。
- **予想廃棄損失** — 商品ごとの予想損失金額と全体の損失規模を合わせて示します。
- **割引・配置の提案** — 割引率や前面配置の有無など、実行方法を提案します。

## 使い方

1. 見切りが必要な商品の推奨を要求します。
2. 主要サマリーで、すぐに進める対象と保留する対象を確認します。
3. 詳細な数値で、商品ごとの廃棄率、直近の売上、在庫を確認します。
4. 提案された割引率と配置で実行します。

## 使い方のヒント

廃棄率が高くても、直近の売上が大きく伸びた商品は、割引よりも発注調整が適している場合があります。詳細な数値を確認し、割引対象と回転調整対象を分けて実行してください。
