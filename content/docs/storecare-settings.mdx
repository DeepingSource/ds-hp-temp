실시간 화면 오른쪽 위의 **설정(⚙️)** 아이콘을 누르면 설정 화면이 열립니다.

![설정 화면](/images/docs/storecare/getting-started/_images/settings.png)

*설정 — 알림 · 계정 · 언어 · 기타*

| 구분 | 항목 | 설명 |
| --- | --- | --- |
| **알림** | 알림 설정 | 어떤 알림을 받을지 관리합니다. |
| **계정** | 내 정보 | 계정 정보를 확인하고 관리합니다. |
| **Language** | 한국어 | 앱 표시 언어를 선택합니다. |
| **기타** | 튜토리얼 다시보기 | 첫 로그인 때 보여 준 화면 안내 튜토리얼을 다시 실행합니다. |
