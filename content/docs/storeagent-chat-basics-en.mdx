AI chat is where you query your store data through conversation. Type a question into the input box at the bottom or press a suggested-question button, and a progress card shows the analysis steps before the answer arrives as a summary and detailed evidence. You can register useful answers as a Task or a memo and carry them through to follow-up action.

## Key features

- **Question input** — enter a question in plain language in the input box at the bottom to request an analysis.
- **Suggested-question buttons** — common questions are offered as buttons you can run right away.
- **Progress card** — shows the answer-generation steps so you know how the analysis is progressing.
- **Summary and detailed answers** — the summary is shown first, and you expand "Details" to check the evidence.
- **Using answers** — register an answer as a Task or a memo and note whether it was helpful.

## How to use it

1. Press a suggested-question button or type a question in the input box at the bottom. (e.g., "Give me this morning's briefing")
2. Wait in the progress card until the analysis steps are complete.
3. Check the summary, and expand "Details" if you need to review the evidence.
4. Register useful answers as a Task or a memo to use in follow-up action.

## Tips

- Asking something like "Give me this morning's briefing" gives you a summary of sales, visits, waste, and repositioning opportunities.
- If an answer is useful, register it as a Task to carry it through as an action item.
