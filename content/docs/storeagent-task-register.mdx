스토어에이전트의 답변은 읽고 끝내는 것이 아니라 실행 과제로 옮길 수 있습니다. 마음에 드는 추천을 'Task로 등록'하면 해야 할 일 목록에 담겨 담당자와 일정을 붙여 관리합니다.

## 주요 기능

- **Task로 등록 버튼** — AI 답변 하단의 버튼으로 답변 내용을 새 과제로 옮깁니다.
- **과제 상세 입력** — 매장, 제목, 실행 시점, 담당자를 지정해 등록합니다.
- **추천 액션 채택** — AI가 제안한 액션 항목을 골라 그대로 과제로 삼을 수 있습니다.

## 사용 방법

1. 실행으로 옮기고 싶은 AI 답변을 확인합니다.
2. 답변 하단의 'Task로 등록'을 누릅니다.
3. 새 Task 등록 창에서 제목과 실행 시점, 담당자를 입력합니다.
4. '등록'을 눌러 과제 목록에 추가합니다.
