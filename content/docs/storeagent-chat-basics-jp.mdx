AI チャットは、店舗データを会話で尋ねる場所です。下部の入力欄に質問を書くか、おすすめの質問ボタンを押すと、進行状況カードが分析の段階を示したあと、要点の要約と詳細な根拠で回答します。役立つ回答は Task やメモとして登録し、後続の対応へつなげられます。

## 主な機能

- **質問の入力** — 下部の入力欄に自然言語で質問を入力して分析を依頼します。
- **おすすめの質問ボタン** — よく使う質問をボタンで用意し、すぐに実行できます。
- **進行状況カード** — 回答生成の段階を表示し、分析の進み具合を知らせます。
- **要点・詳細の回答** — 要点の要約をまず示し、「詳細」を開いて根拠を確認します。
- **回答の活用** — 回答を Task やメモとして登録し、役立ったかどうかを残せます。

## 使い方

1. おすすめの質問ボタンを押すか、下部の入力欄に質問を入力します。（例：「今朝のブリーフィングをして」）
2. 進行状況カードで、分析の段階が完了するまで待ちます。
3. 要点の要約を確認し、必要であれば「詳細」を開いて根拠を見ます。
4. 役立つ回答は Task またはメモとして登録し、後続の対応に活用します。

## 活用のヒント

- 「今朝のブリーフィングをして」のように尋ねると、売上・来店・廃棄・再配置の機会を要約します。
- 回答が役立つ場合は Task として登録し、実行項目へつなげてください。
