Main path analysis is a core feature that automatically extracts the main route (the superhighway) most customers actually follow from the many movement paths in your store.

This route visually shows the "spatial flow with the highest exposure" in the store, and it becomes a strategic anchor for merchandising, promotions, and store path design.

> **💡 팁: Entrance zone required**
>
> Main path analysis requires that an entrance zone is configured in [Zone Config](/en/resources/docs/storeinsight-zones).

## Report layout

### Automatic core-path extraction

- Among customer movement logs, the single most frequently traveled route is visualized on the map.
- You can intuitively understand the traffic frequency and direction of this route.

**Examples**

- Concentrate new and flagship merchandise on the core route to maximize exposure.
- Concentrate marketing resources — POP, promotions — around the core path to maximize ROI.

### Set an origin and view the main-path sequence

- Select an entrance zone from the map or list to extract the main path that starts from that zone.
- The extracted path is provided as an ordered sequence so you can step through the zones the customer traveled.

**Examples**

- Understand the customer purpose and movement pattern by entrance type (front entrance, subway-connected entrance, etc.).
- Design merchandise / VMD strategies separately by entrance (for example, F&B-led vs. fashion-led) for higher efficiency.

### Filter-based main-path analysis

- Filter the main path by gender, age group, dwell time, and other customer attributes for tailored analysis.
- Identify the paths a specific target customer group actually prefers.

**Examples**

- Reinforce related-product displays along the main path of "women in their 20s" (color cosmetics → SPA brand → dessert café).
- Strengthen impulse-purchase displays around the path of "quick-purchase (under-five-minute dwell)" customers (entrance → essentials → Counter).

## Use cases

### Sharper merchandising

- Place core merchandise on the data-driven main path to maximize sales efficiency.
- Validate whether placement of related products translates into actual customer flow, securing core evidence for layout design.

### Crowd and bottleneck response

- Identify the spots where main paths concentrate to predict congestion in advance and design operations.
- Resolve congestion at concentrated points so that customer flow stays smooth and shopping satisfaction increases.

### Marketing and resource concentration

- Optimize the placement of marketing resources — ads, POP, events — along the main path with the largest exposure.
- When inflow into a zone is low, use this as strategic evidence for improving accessibility or repurposing the zone.

### Before-and-after comparison

- **Period comparison** — Compare main paths before and after store renewals, VMD changes, or promotions side by side to visually validate effectiveness.
- **Filter comparison** — Compare main paths between target customer groups (men vs. women, 20s vs. 40s) to set the direction of tailored strategies.

### Combined analytics workflow

Main path analysis works on its own, but synergy grows when combined with other reports. We recommend combining the analyses in this order.

1. **Identify the main path** — Use main path analysis to derive the most-traveled route.
2. **Confirm segment movement and drop-off** — Use [Inter-Zone Traffic](/en/resources/docs/storeinsight-inter-zone-traffic) to verify movement rate and inefficient drop-off rate between segments of the path.
3. **Evaluate zone performance along the path** — Use [Zone interest analysis](/en/resources/docs/storeinsight-zone-interest) to judge whether each zone on the path attracts customer interest (Hot) or has relatively low attention (Cold).
4. **Validate sales contribution** — Use [Purchase conversion analysis](/en/resources/docs/storeinsight-conversion-rate) to compare purchase rates between customers who follow the main path and those who do not, confirming the strategy's effectiveness.

### Advanced filter usage

- Compare the main paths of "quick-purchase (under-five-minute dwell)" customers and "browsers (over-ten-minute dwell)" to design merchandise placement optimized for each group.
- Compare the main paths of your primary target customers with those of non-primary targets to find ways to attract the non-primary group.

> **💡 팁**
>
> Main path analysis provides the most direct answer to the question, "where should we focus to improve sales and experience." Grounded in actual customer movement, it is a strategic map for maximizing in-store space efficiency and managing congestion.
