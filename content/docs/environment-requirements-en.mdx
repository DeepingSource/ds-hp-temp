A guide to the CCTV, network, and dashboard access environment that, when checked before deployment, speeds up the pre-check and installation. The exact minimum specifications vary from store to store, so your account manager will verify them with you during the on-site survey and pre-check stage of deployment.

## Basic Principle

DeepingSource products are built on the basic premise of **using the CCTV and internet connection already in the store as they are**. Rather than installing large amounts of new dedicated equipment, we proceed by verifying whether the existing environment is usable for analysis and supplementing only the parts that need it.

## CCTV and Cameras

- In most cases, we **use the existing installed CCTV as it is**. Installing new cameras for analysis is not required.
- Whether the camera actually covers the area you want to analyze (entrance, shelves, checkout, etc.), and whether the field of view and installation location are appropriate, matters more than image quality.
- If a camera is too old or a certain area is a blind spot, we will advise you during the pre-check on the points that need to be supplemented.

## Network

- An internet connection that can reliably deliver camera footage to the analysis system is required.
- If the store uses a closed network (not connected to the external internet) or has a separate firewall policy, we discuss the integration method separately during the pre-check.
- For multiple stores, the network configuration may differ from store to store, so verification is needed on a per-store basis.

## Dashboard Access

The dashboard (e.g., store insight) is accessed through a **web browser** with no separate program installation. Any PC with a recent version of a common browser (such as Chrome) and an internet connection can use it with no additional preparation.

## POS Integration (Optional)

If you want reports that view sales data alongside metrics such as purchase conversion rate, you can add POS integration. We verify the type of POS system you use and whether data integration is possible during the pre-check, and the detailed procedure will be covered in the POS Integration document.

## Frequently Asked Questions

### How do I check whether our store's CCTV will work?
During the pre-check stage, your account manager checks the number of cameras, field of view, and installation locations, and advises on whether they can be used.

### What about stores with an unstable internet connection?
The integration method can be adjusted depending on the network condition. We verify it together during the pre-check and advise on alternatives.

### Can I view the dashboard on mobile too?
Because it is web-browser based, you can access it from a mobile browser as well. However, the detailed screen layout is optimized for a PC environment.
