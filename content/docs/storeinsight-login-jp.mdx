StoreInsight は Web ブラウザを通じてアクセスします。管理者から発行されたユーザー ID とパスワードでログインしてください。

![StoreInsight ログイン画面](/images/docs/storeinsight/ja/getting-started/_images/login/login-screen.png)

*ログイン画面*

## アクセス URL

ご運営の店舗によってアクセス URL が異なります。発行されたアクセス URL を Web ブラウザのアドレスバーに入力してください。

## ログイン手順

1. **user ID** 入力欄にユーザー ID を入力します。
2. **パスワード** 入力欄にパスワードを入力します。
3. **ログイン** ボタンをクリックすると、ダッシュボードに移動します。

> **💡 팁: ログインできませんか?**
>
> ユーザー ID・パスワードの確認、Caps Lock の状態、ブラウザキャッシュの削除などで解決しない場合は、[FAQ](/jp/resources/docs/storeinsight-faq) の「ログインできない場合」の項目をご確認ください。
