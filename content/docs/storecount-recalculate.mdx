측정 결과가 이상하게 나왔을 때는 현장에 다시 나가지 않아도 됩니다. store count는 측정 시작 시점의 화면과 선을 그대로 저장해 두기 때문에, 저장된 측정만으로 결과를 다시 계산할 수 있습니다. 데이터 화면에서 '라인 다시 맞춰 재계산'을 선택하면 됩니다.

## 주요 기능

- 측정 시작 시점의 화면과 선을 그대로 불러옵니다.
- 선의 위치와 방향을 다시 조정할 수 있습니다.
- 수정한 선을 기준으로 같은 영상을 다시 계산합니다.

## 사용 방법

1. 데이터 화면에서 **'라인 다시 맞춰 재계산'**을 선택합니다. 측정 시작 시점의 화면과 선이 그대로 불려와집니다.
2. 선의 위치나 방향을 고칩니다.
3. **'이 라인으로 계산하기'**를 누릅니다. 수정된 선 기준으로 같은 영상을 다시 계산해 줍니다.

## 사용 팁

- 현장에서 선을 다시 그릴 필요 없이, 저장된 측정만으로 결과를 다시 뽑을 수 있습니다.
- 재계산은 측정 시작 시점에 저장된 선을 기준으로 진행되며, 현재 화면의 선에는 영향을 주지 않습니다.
