스토어인사이트는 웹 브라우저를 통해 접속합니다. 관리자로부터 발급받은 사용자 ID와 비밀번호로 로그인하세요.

![스토어인사이트 로그인 화면](/images/docs/storeinsight/getting-started/_images/login/login-screen.png)

*로그인 화면*

## 접속 주소

운영하시는 매장에 따라 접속 URL이 다릅니다. 발급받은 접속 주소를 웹 브라우저 주소창에 입력해 주세요.

## 로그인 절차

1. **사용자 ID** 입력란에 사용자 ID를 입력합니다.
2. **비밀번호** 입력란에 비밀번호를 입력합니다.
3. **로그인** 버튼을 클릭하면 대시보드로 이동합니다.

> **💡 팁: 로그인이 안되시나요?**
>
> 사용자 ID·비밀번호 확인, Caps Lock 상태, 브라우저 캐시 삭제 등으로 해결되지 않으면 [FAQ](/resources/docs/storeinsight-faq)의 "로그인이 안될 경우" 항목을 확인해 주세요.
