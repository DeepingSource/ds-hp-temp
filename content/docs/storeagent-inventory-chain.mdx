재고가 부족해 결품으로 이어지고, 결품이 다시 매출 손실로 번지는 연쇄를 미리 짚어 줍니다. 결품 위험 품목이 있는지 물으면 위험도 기준으로 점검한 결과를 알려 줍니다.

## 주요 기능

- **결품 위험 점검** — 위험도 기준을 넘는 품목이 있는지 확인해 알림 건수로 정리합니다.
- **용어 명확화** — 등급처럼 뜻이 여러 갈래인 표현은 정확한 제품명 기준으로 다시 확인하도록 안내합니다.
- **선제 점검 안내** — 경보 수준이 아니더라도 판매가 빠른 품목은 미리 재고를 확인하도록 짚어 줍니다.

## 사용 방법

1. 결품이 걱정되는 상황을 채팅창에 묻습니다.
2. 리스크 점검 단계가 위험도 기준으로 품목을 훑습니다.
3. 핵심 요약에서 위험 알림 건수를 확인합니다.
4. "상세"에서 판단 근거와 이어질 위험 신호를 살펴봅니다.

## 사용 팁

- 등급·구간처럼 뜻이 넓은 표현보다 제품명으로 물으면 더 정확히 조회됩니다.
- 위험 알림이 없어도 회전이 빠른 품목은 주기적으로 재고를 확인하세요.
