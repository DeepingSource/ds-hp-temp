선 설정이 끝나면 우측 상단의 **측정 시작** 버튼을 누릅니다. 상태 표시가 '측정 중'으로 바뀌고, 옆에 측정 시간이 흐르기 시작합니다.

## 주요 기능

측정 중에는 두 카드에서 실시간 숫자를 확인할 수 있습니다.

- **유동인구** — 지금까지 매장 앞을 지나간 사람 수입니다. 하늘색 선 2개 이상을 통과한 사람과 분홍색 선을 방향에 따라 통과한 사람을 합해 셉니다.
- **유입 · 전환율** — 유입은 매장 안으로 들어온 사람 수(분홍색 선 통과)이며, 전환율은 지나간 사람 중 실제 들어온 비율입니다.
- 각 카드의 **i** 버튼을 누르면 그 숫자가 어떻게 집계되는지 설명을 볼 수 있습니다.

## 사용 방법

1. **측정 시작** — 우측 상단 버튼을 누르면 상태가 '측정 중'으로 바뀌고 시간이 흐릅니다.
2. **일시정지** — 잠시 멈추는 것입니다. '다시 측정'을 누르면 멈췄던 시점부터 이어집니다. 점심시간 등 잠깐 중단할 때 씁니다.
3. **완료** — 측정을 완전히 마무리하는 것입니다. 완료해야 결과를 받을 수 있고, 한 번 완료한 측정은 끝납니다. '완료'를 누르면 측정 장소 이름을 적는 칸이 나오는데, '정문 측정', '6월 평일'처럼 적어 두면 나중에 데이터를 구분하기 좋습니다. 비워 두어도 됩니다.
4. **이어서 측정** — 이미 완료한 측정에 새 측정을 더해 한 묶음으로 합치는 기능입니다. 오전 측정을 완료한 뒤 오후에 '이어서 측정'으로 시작하면 오전 숫자에 오후 숫자가 더해지고 시간도 이어집니다. 완전히 새로 시작하려면 '새 측정 시작'을 고릅니다.

## 사용 팁

- 측정 중에는 선을 수정·삭제할 수 없습니다.
- 측정 중에는 카메라와 AI 박스를 움직이지 마세요. 화면이 바뀌면 측정이 어긋납니다.
