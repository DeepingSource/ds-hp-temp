매장의 각 구역이 지금 얼마나 붐비는지 실시간으로 보여 줍니다. 구역 이름을 물으면 현재 혼잡 수준과 인력 권고를 알려 주고, 카메라 화면을 함께 살펴볼 수 있습니다.

## 주요 기능

- **구역별 혼잡도** — 특정 구역의 현재 고객 수와 혼잡 수준을 실시간으로 알려 줍니다.
- **핵심 지표 카드** — 현재 고객, 직원, 평균 체류, 전주 대비를 한 화면에 정리합니다.
- **LIVE 카메라 피드** — 여러 채널의 현장 화면을 짧은 주기로 갱신해 함께 보여 줍니다.

## 사용 방법

1. 궁금한 구역 이름을 채팅창에 묻습니다.
2. 실시간 혼잡도 분석 단계가 현재 상태를 확인합니다.
3. 핵심 요약에서 혼잡 수준과 인력 권고를 확인합니다.
4. 오른쪽 지표 카드와 LIVE 피드로 현장을 직접 살펴봅니다.

## 사용 팁

- 혼잡도는 짧은 시간에도 변하므로 잠시 더 지켜본 뒤 인력을 조정하세요.
- 전주 대비 수치로 지금이 평소보다 붐비는지 한산한지 가늠할 수 있습니다.
