## ログインできない場合

- ユーザー ID とパスワードが正しいか再度ご確認ください。
- キーボードの Caps Lock キーがオフになっているかご確認ください。
- ご利用中の Web ブラウザのキャッシュを削除してから再度お試しください。

## データが表示されない場合

- レポートの表示期間が正しく設定されているかご確認ください。
- 適用されているフィルターを初期化してから再度お試しください。
- ページを再読み込み(F5)してみてください。

## パフォーマンスに関する問題

- Web ブラウザを最新バージョンにアップデートしてください。
- 不要に開かれている他のブラウザタブを閉じてください。
- インターネット接続が安定しているかご確認ください。

---

解決しない問題は、**システム管理者** または **Deeping Source カスタマーサポート** までお問い合わせください。
