**실시간** 탭은 "지금 이 순간" 우리 매장의 상태를 보여 줍니다. 화면 맨 위에는 구역 상태 요약과 매장 평면도가 있습니다.

![실시간 매장 상태 요약과 평면도](/images/docs/storecare/live/_images/status.png)

*구역 상태 요약 칩과 매장 평면도(미니맵)*

## 구역 상태 요약

매장의 각 구역(진열대·테이블·냉장 설비 등)은 현재 상태에 따라 네 단계로 분류되고, 각 단계에 해당하는 구역이 몇 개인지 칩으로 표시됩니다.

| 상태 | 색 | 의미 |
| --- | --- | --- |
| **완벽** | 🔵 파랑 | 기준을 완전히 충족한 상태 |
| **좋음** | 🟢 초록 | 대체로 양호한 상태 |
| **주의** | 🟡 노랑 | 곧 정리가 필요한 상태 |
| **나쁨** | 🔴 빨강 | 즉시 조치가 필요한 상태 |

평면도 위에서도 각 구역이 같은 색으로 칠해져, 어느 위치의 어떤 구역이 문제인지 바로 알 수 있습니다.

> **📌 참고**
>
> 연결 오류 등으로 상태 정보를 받지 못한 구역은 색이 채워지지 않은 '데이터 없음'으로 표시됩니다. 데이터 없음이 하루 이상 이어지면 문의해 주세요.

## 구역 눌러서 보기

평면도에서 구역을 누르면 아래에서 해당 구역의 카드가 올라옵니다.

![구역 상태 카드](/images/docs/storecare/live/_images/zone-status.png)

*구역을 누르면 현재 상태와 지속 시간이 표시됩니다*

- 구역 이름과 **현재 상태**, 그 상태가 **얼마나 지속되고 있는지**(예: `좋음 9시간 5분째`)를 보여 줍니다.
- **타임라인 보기**를 누르면 해당 구역의 지난 24시간 상태 변화를 [타임라인](/resources/docs/storecare-timeline)에서 확인할 수 있습니다.

## 상태 요약 한마디

평면도 아래에는 오늘 매장 상태를 한 문장으로 정리한 요약 카드가 있습니다. 표정 아이콘과 함께 "전반적으로 잘 유지되고 있어요"처럼, 지금 매장이 어떤 분위기인지 빠르게 알려 줍니다.
