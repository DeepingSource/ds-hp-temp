본격적으로 시작하기 전에 store count의 측정 화면 구성을 한 번만 살펴봅니다. 지금은 '이런 게 있구나' 정도만 알아두면 됩니다. 모든 조작은 핸드폰 화면에서 이뤄집니다.

## 주요 기능

1. **상단 버튼 줄** — 측정을 시작·일시정지·완료하고, 데이터를 받는 버튼이 모여 있습니다.
2. **상태 표시** — '측정 준비됨' / '측정 중' / '측정 완료'와 측정 시간을 영상 왼쪽 위에 보여줍니다.
3. **영상 화면** — 카메라에 보이는 모습입니다. 이 위에 손가락으로 기준선을 긋습니다.
4. **유동인구 / 유입 선택** — 어떤 종류의 기준선을 그릴지 고르는 곳입니다.
5. **라인 목록** — 지금까지 그린 기준선 목록입니다. 이름 변경·방향 전환·삭제를 여기서 합니다.

## 사용 팁

- 측정 중에는 '유동인구·유입 선택'과 '라인 목록'이 흐리게 잠깁니다. 실수로 선을 건드려 숫자가 틀어지는 것을 막기 위한 안전장치입니다.
- 기준선을 새로 그리거나 고치려면 측정을 완료하거나 시작 전에 작업하세요.
- 상태 표시의 측정 시간을 보면 지금 얼마나 오래 측정하고 있는지 바로 확인할 수 있습니다.
