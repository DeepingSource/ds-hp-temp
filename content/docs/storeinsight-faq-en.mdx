## If you can't log in

- Double-check that your user ID and password are correct.
- Make sure the Caps Lock key on your keyboard is off.
- Clear your web browser's cache and try again.

## If data is not displayed

- Check that the report's search period is set correctly.
- Reset any applied filters and try again.
- Refresh the page (F5).

## Performance issues

- Update your web browser to the latest version.
- Close any unnecessary browser tabs.
- Make sure your internet connection is stable.

---

If the issue persists, please contact your **system administrator** or **Deeping Source customer support**.
