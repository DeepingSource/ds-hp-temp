This guide covers the status of the public API and the options available to you now when you need data integration.

## Public API in Preparation

Currently, DeepingSource **does not provide a separate publicly available API.** Opening up capabilities such as report data retrieval in API form is an item on our roadmap, and as soon as it is ready, this document will provide the specific endpoints and authentication methods.

## For Now

For now, the standard approach is to log in to the dashboard and view reports directly. If you want to view sales data as well, see the POS Integration document; if multiple people need to access it together, see the Dashboard Access document.

## What This Will Cover (Planned)

Once the public API is ready, this document plans to focus on how to programmatically retrieve the metrics you check in the dashboard, such as report data. The exact scope and schedule have not yet been finalized.

## Data Integration Inquiries

If you have a situation that requires an API — such as needing to integrate report data into an in-house system — let your account manager know your requirements. Even before the public API launch, depending on the situation we may be able to discuss a separate integration method.

## Frequently Asked Questions

### Is there a release schedule for the API?
There is no finalized schedule yet. As soon as it is ready, we will provide notice through this document and announcements.

### There's no API — is there a way to retrieve data automatically?
Depending on the situation, you can discuss a separate integration method with your account manager. Please inquire with the data you need and its intended use.
