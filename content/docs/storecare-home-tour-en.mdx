After logging in, the **Real-time** screen opens first. It's made up of a top bar, a content area in the middle, and navigation at the bottom.

![Real-time home screen, top](/images/docs/storecare/en/getting-started/_images/home-overview.png)

*Top bar: last-updated time and settings, with the store selector below*

## Top bar

- **Last updated** — the top left shows `Mon DD, HH:MM Updated`. Tap the refresh icon beside it to reload the latest status.
- **Settings (⚙️)** — the gear icon at the top right opens the [Settings](/en/resources/docs/storecare-settings) screen.

## Store selector

The name of the store you're viewing appears below the top bar (e.g. **Daechi StoreCare**). If you run multiple stores, use the left/right arrows or tap the store name to pick a store from the list.

![Store settings sheet](/images/docs/storecare/en/getting-started/_images/store-switcher.png)

*Tap the store name to switch between stores*

## Bottom navigation

Three tabs at the very bottom move you between the main screens.

| Tab | Description |
| --- | --- |
| **Real-time** | Check your store's status right now, live. |
| **Report** | Pick a past date and look back at that day. |
| **Inbox** | See store alerts in one place. The unread count shows as a badge. |

> **💡 Tip**
>
> See each tab's full details in the [Real-time](/en/resources/docs/storecare-live), [Report](/en/resources/docs/storecare-report), and [Inbox](/en/resources/docs/storecare-notification) sections in the left menu.
