대시보드는 매장 내 방문자 동선, 인구통계학적 특성, 그리고 실시간 체류 현황을 직관적으로 확인할 수 있는 운영 도구입니다. 실시간 데이터와 당일 통계 분석을 결합하여, 빠른 의사결정과 현장 대응을 지원합니다.

## 1. 주요 기능

### 1.1 방문자 동선 추적

- **설명:** 매장 내부에서 방문자의 이동 경로를 실시간으로 확인합니다.
- **활용 예시:** 혼잡 구역 파악, 상품 진열 최적화, 직원 배치 조정.

### 1.2 성별 / 연령대 필터

- **설명:** 방문자 데이터를 성별 또는 연령대별로 필터링하여 조회할 수 있습니다.
- **활용 예시:** 특정 타겟 고객층의 행동 패턴 분석, 맞춤형 프로모션 기획.

### 1.3 지도 표시

- **설명:** 매장 지도 위에 방문자의 현재 위치를 표시합니다.
- **활용 예시:** 혼잡 구간 모니터링, 구역별 체류 인원 실시간 확인.

## 2. 금일 통계 정보

### 2.1 체류자 수

- **설명:** 현재 매장에 머무르고 있는 방문자 수를 실시간으로 표시합니다.
- **활용 예시:** 매장 혼잡도 판단, 안전 관리 기준 대응.

### 2.2 남:여 성비

- **설명:** 현재 매장 내 남성과 여성 방문자의 비율을 제공합니다.
- **활용 예시:** 성별 비율 변화에 따른 마케팅 메시지 조정.

### 2.3 주 연령층

- **설명:** 오늘 방문자 중 가장 많은 비중을 차지하는 연령대를 표시합니다.
- **활용 예시:** 연령대 맞춤 상품 배치, 프로모션 타겟 선정.

## 3. 오늘의 방문자 분석

### 3.1 총 방문자 수

- **설명:** 오늘 하루 동안의 누적 방문자 수를 제공합니다.
- **활용 예시:** 일별 유입량 비교, 특별 행사 효과 측정.

### 3.2 시간대별 방문자 그래프

- **설명:** 하루 동안의 시간대별 방문자 수 변화를 그래프로 표시합니다.
- **활용 예시:** 피크타임 파악, 근무 교대 시간 최적화.

### 3.3 성별 및 연령 분포

- **설명:** 방문자의 성별 및 연령대 분포를 시각화된 차트로 제공합니다.
- **활용 예시:** 고객 세그먼트 분석, 타겟 마케팅 전략 수립.

## 4. 사용 팁

1. **필터 기능 적극 활용:** 성별·연령대 필터를 사용하면 특정 고객군의 행동 패턴을 쉽게 파악할 수 있습니다.
2. **실시간·누적 데이터 비교:** 실시간 현황과 누적 데이터를 함께 참고하면 정확한 현장 판단이 가능합니다.
3. **이벤트·프로모션 연계 분석:** 특정 이벤트 진행 시간대와 방문자 변화 추이를 비교해 효과를 평가할 수 있습니다.
