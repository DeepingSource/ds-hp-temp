바로 실행할 일은 아니지만 기억해두고 싶은 답변은 메모로 저장할 수 있습니다. 'Task로 등록'이 해야 할 일을 관리한다면, '메모로 등록'은 참고할 내용을 남겨두는 기능입니다.

## 주요 기능

- **메모로 등록 버튼** — AI 답변 하단의 버튼으로 답변을 메모로 저장합니다.
- **점장 메모 목록** — 저장한 메모는 날짜와 함께 메모 목록에 쌓입니다.
- **직접 메모 추가** — AI 답변 외에 필요한 내용을 메모 입력란에 직접 남길 수 있습니다.

## 사용 방법

1. 남겨두고 싶은 AI 답변을 확인합니다.
2. 답변 하단의 '메모로 등록'을 누릅니다.
3. 저장된 메모를 점장 메모 목록에서 확인합니다.
4. 필요하면 메모 입력란에 내용을 직접 추가합니다.

## 사용 팁

메모는 실행 여부를 따지지 않고 자유롭게 모아두는 공간입니다. 나중에 실제로 해야 할 일이라면 Task로, 단순 참고라면 메모로 나눠 쓰면 두 기능을 구분해 활용하기 좋습니다.
