saai count는 복잡한 공사 없이 AI 박스와 모바일 설정만으로 측정을 시작할 수 있습니다. 설치부터 첫 데이터 확인까지 대략 5분이면 준비됩니다.

## 주요 기능

- **간편 설치** — AI 박스를 측정할 위치에 두고 전원과 네트워크만 연결합니다.
- **모바일 설정** — 휴대폰으로 측정 구역과 방향을 설정합니다.
- **빠른 시작** — 설정을 마치면 곧바로 유동·유입 집계가 시작됩니다.

## 사용 방법

1. AI 박스를 매장 앞이 잘 보이는 위치에 설치합니다.
2. 전원과 네트워크를 연결합니다.
3. 모바일에서 측정 구역과 방향을 지정하고 저장합니다.
4. 잠시 후 첫 측정 데이터가 들어오는지 확인합니다.

## 사용 팁

- 매장 앞 통행 동선이 화면에 고르게 담기도록 박스 방향을 맞추면 측정 정확도가 높아집니다.
- 설치 후 첫날은 데이터가 적으니, 며칠 쌓인 뒤 지표를 해석하세요.
