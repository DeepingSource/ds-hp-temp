## 스토어인사이트가 제공하는 가치


**효율적인 운영**

인기 구역엔 자원을 더 배치하고, 비효율 구역은 개선합니다.
**성과 높은 마케팅**

성별·연령별 관심 구역에 맞춘 타겟 캠페인을 설계합니다.
**고객 경험 향상**

최적화된 동선 설계로 쇼핑 피로도를 줄입니다.
**데이터 기반 의사결정**

실제 데이터를 근거로 더 빠르고 정확한 판단을 내립니다.


## 가이드 구성

- **시작하기** — 로그인, 대시보드, 데이트 피커, 데이터 필터 사용법
- **대시보드 설정** — 라인·구역·행사 일정·광고 성과 설정
- **리포트** — 분석 리포트 사용 매뉴얼 (방문자 분석, 히트맵, 동선, 퍼널, 구매 전환 분석 등)
- **문제 해결 / FAQ** — 자주 묻는 질문과 해결 방법
