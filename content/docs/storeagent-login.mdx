스토어에이전트에 로그인한 뒤 상단바에서 분석할 매장을 선택합니다. 여러 매장을 운영한다면 상단바의 매장 스위처로 다른 매장으로 빠르게 전환할 수 있습니다. 선택한 매장 기준으로 채팅 답변과 오버뷰 지표가 함께 갱신됩니다.

## 주요 기능

- **매장 선택 드롭다운** — 상단바 왼쪽의 매장 이름을 눌러 담당 매장을 선택합니다.
- **매장 전환** — 여러 매장을 운영할 때 드롭다운에서 다른 매장으로 바로 전환합니다.
- **사용자 정보 표시** — 상단바 오른쪽에 현재 로그인한 계정과 권한이 표시됩니다.
- **로그아웃** — 상단바 오른쪽 끝의 버튼으로 안전하게 로그아웃합니다.

## 사용 방법

1. 계정 정보로 로그인합니다.
2. 상단바 왼쪽의 매장 이름 드롭다운을 엽니다.
3. 목록에서 확인할 매장을 선택하면 화면 전체가 해당 매장 기준으로 갱신됩니다.
4. 사용을 마치면 오른쪽 끝의 로그아웃 버튼을 누릅니다.

## 사용 팁

- 매장을 전환하면 채팅 답변과 오버뷰 지표가 모두 선택한 매장 기준으로 바뀝니다.
- 여러 매장을 담당한다면 분석을 시작하기 전에 대상 매장이 맞는지 먼저 확인하세요.
