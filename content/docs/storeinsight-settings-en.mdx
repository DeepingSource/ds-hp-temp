Configure the building blocks that StoreInsight uses to measure customer behavior. Set up lines, zones, event schedules, and ad-viewing analysis at install time — every report's quality depends on these settings.


- [Line Config](/en/resources/docs/storeinsight-lines) — Draw virtual lines to count passing customers
- [Zone Config](/en/resources/docs/storeinsight-zones) — Define passage, interest, counter, display stand, and other zones
- [Event scheduling](/en/resources/docs/storeinsight-events) — Sync Google Calendar for per-event analytics
- [Ad-viewing analysis](/en/resources/docs/storeinsight-ad-tracking) — Mark ad locations and viewing zones
