公開APIの現状と、データ連携が必要なときに今確認できる方法をご案内します。

## 公開APIは準備中

現在、DeepingSourceは**外部に公開されたAPIを別途提供していません。**レポートデータの取得などをAPIの形で開放することはロードマップにある項目で、準備でき次第、このドキュメントで具体的なエンドポイントと認証方式をご案内する予定です。

## 今はこのように

今は、ダッシュボードにログインしてレポートを直接確認するのが基本的な方法です。売上データも合わせて見たい場合は「POS連携」を、複数の担当者が一緒にアクセスする必要がある場合は「ダッシュボードへのアクセス」ドキュメントをご参照ください。

## 今後扱う内容（予定）

公開APIが準備できたら、このドキュメントでは、レポートデータの取得のように、ダッシュボードで確認する指標をプログラムで取得する方法を中心に扱う計画です。正確な範囲とスケジュールはまだ確定していません。

## データ連携のお問い合わせ

社内システムにレポートデータを連携する必要があるなど、APIが必要な状況であれば、担当マネージャーに要件をお知らせください。公開APIのリリース前でも、状況に応じて別途連携方法を協議できる場合があります。

## よくある質問

### APIのリリーススケジュールはありますか？
まだ確定したスケジュールはありません。準備でき次第、このドキュメントとお知らせを通じてご案内いたします。

### APIがないのですが、データを自動で取得する方法はありますか？
状況に応じて、担当マネージャーと別途連携方法を協議できます。必要なデータと用途をお問い合わせください。
