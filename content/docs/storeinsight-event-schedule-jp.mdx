- 登録されたイベント期間中の訪問・滞在・関心度データを自動的に集計します。
- 2つのイベントを同時に選択して比較すれば、イベント別の効果を定量的に評価できます。


> **📌 참고**
>
> **Add-on レポートです。**

## 前提条件

- [イベントのスケジュール設定](/jp/resources/docs/storeinsight-events) で Google カレンダーが連携されている必要があります。

イベント日程は、店舗で実施されるイベント別の顧客流入および人口統計学的反応を精密に分析する機能です。どの顧客層がどのイベントに最も関心を示し訪問したかを把握し、イベント戦略をデータに基づいて最適化できます。

## 主要機能

### イベント訪問者統計

- 選択したイベント期間中の顧客流入および行動データを総合集計します。
- 総訪問者、日別/時間帯別の流入推移、性別/年齢分布などの主要な基本統計を提供します。

**活用例**

- 時間帯別の訪問者を把握し、人員および在庫配置を最適化
- 性別および年齢分布を把握し、次回イベントの企画に反映

### イベント成果の比較分析

- 運営期間内のイベントから最大2つを選択し、主要指標を比較分析できます。
- 2つのイベントの総訪問者、顧客反応データに基づき、相対的な成果を測定します。

**活用例**

- 「割引」 vs 「ノベルティ」などプロモーション方式別の顧客流入効果の比較
- イベント訪問顧客と主要商品コーナー訪問顧客数の比較分析

### 観測位置の出入り分析

- イベントを選択した後、店舗内の特定の「観測位置」を指定して分析を細分化できます。
- イベント期間中、当該位置を訪れた顧客の出入り統計と人口統計情報を分析します。

**活用例**

- POP/サイネージ近隣の観測位置を訪れた顧客の性別および年齢層の把握
- イベントが主要商品コーナーなど特定空間への流入に与えた影響の分析

## 主要指標 (KPI)

| 指標名 | 説明 | 活用インサイト |
| ------ | ---- | ------------- |
| 総訪問者数 | イベントに流入した全顧客規模 | イベント集客力および初期成果の判断 |
| 人口統計分布 | 訪問顧客の性別および年齢層比率 | ターゲット顧客層の適合性確認 |
| 時間帯別集中度 | 顧客流入が最も多いピーク時間帯 | 人員および在庫配置の効率化 |
| イベント比較成果 | 複数イベント間の総訪問者数、流入効率の差異 | 最適なプロモーション方式の選定 |
| 観測位置流入率 | イベント接点(POP、売り場)を訪れた顧客比率 | イベントディスプレイの実質的効果測定 |

## 活用事例

### マーケティング投資対成果(ROI)の最大化

- 成功イベントの要因分析による最適な集客戦略の確保
- 成果比較分析で非効率なマーケティング要素を除去
- 予算浪費の最小化およびマーケティング投資効率の増大

### マーケティングターゲットの精度向上

- 実際の反応顧客層の分析により次回イベントのターゲットを精緻化
- 顧客層別の好みのイベント把握によるカスタマイズマーケティングの実行

### 店舗運営効率および顧客体験の改善

- 時間帯別混雑度の予測による人員/在庫配置の最適化
- 観測位置分析によるイベント接点(売り場、POP)の効果最大化
- 効率的な混雑管理による顧客離脱率の減少およびショッピング体験の改善

## 上級活用ヒント

> **💡 팁**
>
> **ターゲット顧客向けカスタマイズ戦略** — 実際のイベント反応顧客層(性別/年齢)データを活用し、次回イベントのターゲット顧客群を明確化、流入が高かったメッセージ/プロモーションパターンを抽出してターゲット顧客に最も効果的なマーケティングを実行します。

> **💡 팁**
>
> **現場配置の効率最大化** — 観測位置分析を通じて POP・売り場など現場接点の集客効果を検証し、顧客流入効率の高い接点パターンを把握して現場イベント/ディスプレイの配置に即時適用します。

> **💡 팁**
>
> **予算浪費の防止と成果の強化** — 成功/失敗イベントの成果比較分析を通じて、効率の低いマーケティングタイプに予算が浪費されることを防止し、成功イベントの時間帯パターン分析で人員および在庫投入時点を精緻化します。

## まとめ

> イベント日程は、店舗イベントの顧客流入成果と反応を定量的に比較するツールです。
> データに基づきイベント効率を測定し、次回のイベント企画および店舗運営の最適化に活用されます。
