스토어케어 앱을 열면 로그인 화면이 나타납니다. 가입할 때 등록한 **이메일**과 **비밀번호**를 입력하고 **로그인**을 누릅니다.

![스토어케어 로그인 화면](/images/docs/storecare/getting-started/_images/login.png)

*로그인 화면 — 이메일과 비밀번호 입력*

- **이메일 · 비밀번호** — 두 칸을 모두 채우면 로그인 버튼이 활성화됩니다. 비밀번호 칸 오른쪽의 눈 아이콘으로 입력한 값을 확인할 수 있습니다.
- **비밀번호 변경** — 비밀번호를 잊었거나 바꾸고 싶을 때 사용합니다.
- **회원가입** — 아직 계정이 없는 경우 가입을 진행합니다.

:::tip[첫 로그인 안내]
처음 로그인하면 주요 화면을 짚어 주는 짧은 튜토리얼이 단계별로 표시됩니다. **건너뛰기**로 넘길 수 있고, 나중에 다시 보려면 [설정](/resources/docs/storecare-settings) → **튜토리얼 다시보기**를 누릅니다.
:::
