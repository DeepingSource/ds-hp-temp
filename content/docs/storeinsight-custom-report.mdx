*커스텀 리포트 데모*


커스텀 리포트는 사용자가 원하는 조건에 맞춰 **분석 기준, 필터, 비교 항목을 직접 설정해** 하나의 보고서로 통합 시각화할 수 있는 **맞춤형 리포트 생성 도구**입니다.

이 기능은 반복적인 보고 업무를 줄이고, 실무자가 **목적 중심으로 데이터에 접근**할 수 있게 설계되었습니다.

## 리포트 화면 / 구성

![커스텀 리포트 화면](/images/docs/storeinsight/reports/customization/_images/custom-report/01.png)

*원하는 분석 모듈과 조건을 자유롭게 조합하는 맞춤형 리포트*

![커스텀 리포트 통합 시각화](/images/docs/storeinsight/reports/customization/_images/custom-report/02.png)

*여러 지표와 시각화 모듈을 하나의 보고서로 통합 확인*

### 분석 항목 선택

- 원하는 분석 모듈(예: 방문자 분석, 구역별 분석, 퍼널 분석 등)을 선택해 **조합**할 수 있습니다.
- 하나의 리포트에서 여러 지표·시각화 모듈을 **한눈에 통합 확인**할 수 있습니다.
- **활용 예시**
  - 히트맵 + 대표 동선 + 전환율을 하나의 보고서에 배치
  - 특정 구역 기준으로 체류 시간, 유입 흐름, 세그먼트별 행동을 동시에 분석

### 조건 설정 및 필터

- 날짜 범위 설정(일·주·월·커스텀 기간)
- 특정 구역, 상품군, 고객군(성별·연령대 등) 기준 필터
- 시간대, 재방문 여부, 체류 시간 조건 등 고급 필터링
- **활용 예시**
  - "6월 한 달 동안 20대 여성의 체류 5분 이상 행동만 보기"
  - "이벤트 기간 중 계산대 도달률이 높은 대표 동선 추적"

### 비교 및 시계열 기능

- **전·후 비교:** A/B 테스트, 프로모션 효과, 레이아웃 변경 전후 성과 확인
- **지점 간 비교:** 다점포 운영 시 동일 리포트 기준으로 매장 성과 비교
- **기간별 트렌드:** 전환율·체류 시간 등 주요 지표의 추이 확인

## 활용 사례

### 마케팅 캠페인 성과 리포트

- **분석 항목:** 방문자 수, 체류 시간, 구매 전환율
- **필터:** 캠페인 기간 + 타겟 고객군
- **비교:** 캠페인 전·후

→ "고객 유입은 늘었지만 전환율은 하락 → 리플렛 개선 필요"와 같은 인사이트를 도출할 수 있습니다.

### 매대 리뉴얼 효과 분석

- **분석 항목:** 구역별 유입, 체류, 픽업, 구매
- **필터:** 특정 상품 구역
- **비교:** 리뉴얼 전후 기간

→ "체류 시간은 증가했지만 픽업률은 감소 → 진열 방식 점검 필요"라는 진단이 가능합니다.

### 정기 보고 자동화

- **주간 보고서:** 매주 같은 형식으로 핵심 지표를 묶어 보고
- **경영진 요약본:** 임원 보고용 한 페이지 요약 자동 생성
- **조건별 비교 분석:** 비 오는 날 vs 맑은 날, 이벤트 기간 vs 평소 등

> **💡 팁: 요약**
>
> 커스텀 리포트는 정해진 리포트가 아닌, **내가 보고 싶은 방식대로 분석을 설계하는 도구**입니다.
> 반복되는 분석 작업을 자동화하고, 실무자 중심의 의사결정을 가능하게 합니다.
