스토어에이전트는 매장 데이터를 대화로 물어보는 매장 운영 코파일럿입니다. 채팅으로 질문하면 방문·매출·재고 데이터를 분석해 바로 답합니다. 처음 접속하면 왼쪽에 채팅 시작 화면이, 오른쪽에 매장 오버뷰 지표가 함께 나타납니다.

## 주요 기능

- **대화형 질의응답** — 자연어로 질문하면 매장 데이터를 분석해 답변합니다.
- **추천 질문** — 자주 쓰는 질문을 버튼으로 제공해 클릭만으로 시작할 수 있습니다.
- **매장 오버뷰** — 화면 오른쪽에 고객·매장 지표를 상시 표시합니다.
- **기간 필터** — 평일·주말·최근 2주·전월 기준으로 데이터를 바꿔 봅니다.

## 사용 방법

1. 로그인 후 상단바에서 담당 매장을 선택합니다.
2. 왼쪽 "무엇이 궁금하세요?" 아래의 추천 질문을 누르거나, 하단 입력창에 질문을 직접 입력합니다.
3. 오른쪽 오버뷰에서 매장 지표를 함께 확인하며 후속 질문을 이어갑니다.

## 사용 팁

- 추천 질문으로 시작하면 어떤 분석이 가능한지 빠르게 파악할 수 있습니다.
- 답변을 받은 뒤 이어서 질문하면 맥락을 유지한 채 더 깊이 살펴볼 수 있습니다.
