The charts in the Overview are not just graphs to look at. Tap the chart for a metric you care about and a question about that metric is generated automatically, with an analysis answer following right there. A question that comes to mind while reading the table can be turned into a query on the spot.

## Key Features

- **Chart-click question generation** — Tap a metric chart in the Overview and a question about that metric is entered into the chat box automatically.
- **Streaming answers** — Each step of the process is shown one by one, and the key summary and detailed analysis fill in order.
- **Related analysis links** — Related metrics worth reviewing appear side by side below the expanded chart, so you can keep exploring.

## How to Use

1. Find the chart for the metric you want to review in the Overview.
2. Tap the chart to open the detail view, where a related question is prepared.
3. Tap "Ask in chat again" to get an answer, or refine the question before sending it.
4. When the process steps finish, check the key summary and the details.

## Tips

- If you are curious about a specific date, hover over the chart to check the value before moving on to the question.
- Add your store's situation to the auto-generated question to get a more tailored answer.
