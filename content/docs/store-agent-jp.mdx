## store agent ができること

store agent は、店舗データを代わりに読み取り、次の一手を提案する**対話型の運営コパイロット**です。売上・来店・在庫・スタッフ・CCTV のデータを一か所で尋ね、答えを受け取ってそのまま実行に移せます。

**尋ねると画面が開きます**

「今朝のブリーフィングをして」「弁当の発注を提案して」のように自然言語で尋ねると、それに合った分析画面が開きます。

**根拠までさかのぼれます**

提案には必ず根拠が付きます。「なぜこの数量なのか」を表とデータで確認してから判断してください。

**判断するのは人です**

提案するのは AI、判断するのはオーナーです。気に入った回答は Task やメモとして登録し、実行へつなげます。

## このマニュアルの構成

- **はじめに** — ログイン、店舗の選択、オーバービュー画面、AI チャットの使い方
- **分析の活用** — チャートへの質問、異常検知、発注・棚・スタッフ分析、外部コンテキスト
- **製品マニュアル** — Task・メモの登録、応答の停止、よくある質問
