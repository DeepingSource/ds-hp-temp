AI 박스는 카메라 영상을 분석해 사람 수를 세는 store count의 분석 장비입니다. 매장 안에 두고 전원 콘센트에 연결하기만 하면 됩니다. 별도의 키보드나 마우스는 필요 없습니다.

## 설치 단계

1. 매장 안에서 **30cm × 20cm** 정도의 공간과 **전원 콘센트 1구**를 찾습니다.
2. AI 박스의 전원 콘센트를 연결하면 **자동으로 켜집니다.** 별도의 키보드나 마우스는 필요 없습니다.
3. 카메라와 AI 박스는 **무선**(Wi-Fi)으로 자동 연결됩니다.

## 사용 팁

- AI 박스는 매장 안 어디든 전원만 있으면 됩니다. 카메라와 너무 멀지 않게 두세요.
- 카메라와의 무선 연결은 자동으로 이뤄지므로 별도 설정이 필요 없습니다.
- 전원이 켜진 뒤에는 다음 단계인 핸드폰 연결로 넘어가면 됩니다.
