질문이 잘못됐거나 답변이 예상보다 오래 걸릴 때, 진행 중인 응답을 중간에 멈출 수 있습니다. 응답 생성 중에는 입력창의 전송 버튼이 중단 버튼으로 바뀝니다.

## 주요 기능

- **응답 중단 버튼** — 답변 생성 중 입력창의 버튼을 눌러 진행을 멈춥니다.
- **중단 상태 표시** — 멈추면 '사용자가 응답을 중단했습니다'로 표시됩니다.
- **바로 재질문** — 중단한 뒤 질문을 고쳐 다시 물어볼 수 있습니다.

## 사용 방법

1. AI가 답변을 생성하는 동안 진행 상황을 확인합니다.
2. 입력창의 중단 버튼을 누릅니다.
3. 응답이 멈추고 중단되었다는 안내가 표시됩니다.
4. 질문을 다듬어 다시 입력합니다.

## 사용 팁

답변 방향이 처음부터 어긋났다면 끝까지 기다리기보다 일찍 중단하고 질문을 구체적으로 바꾸는 편이 빠릅니다. 중단해도 이전 대화 내용은 그대로 남습니다.
