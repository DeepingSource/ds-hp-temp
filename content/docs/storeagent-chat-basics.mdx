AI 채팅은 매장 데이터를 대화로 물어보는 공간입니다. 하단 입력창에 질문을 적거나 추천 질문 버튼을 누르면, 진행 상황 카드가 분석 단계를 보여준 뒤 핵심 요약과 상세 근거로 답변합니다. 유용한 답변은 Task나 메모로 등록해 후속 조치로 이어갈 수 있습니다.

## 주요 기능

- **질문 입력** — 하단 입력창에 자연어로 질문을 입력해 분석을 요청합니다.
- **추천 질문 버튼** — 자주 쓰는 질문을 버튼으로 제공해 바로 실행할 수 있습니다.
- **진행 상황 카드** — 답변 생성 단계를 표시해 분석 진행 상태를 알려줍니다.
- **핵심·상세 답변** — 핵심 요약을 먼저 보여주고, '상세'를 펼쳐 근거를 확인합니다.
- **답변 활용** — 답변을 Task나 메모로 등록하고 도움 여부를 남길 수 있습니다.

## 사용 방법

1. 추천 질문 버튼을 누르거나 하단 입력창에 질문을 입력합니다. (예: "오늘 아침 브리핑 해줘")
2. 진행 상황 카드에서 분석 단계가 완료될 때까지 기다립니다.
3. 핵심 요약을 확인하고, 필요하면 '상세'를 펼쳐 근거를 살펴봅니다.
4. 유용한 답변은 Task 또는 메모로 등록해 후속 조치에 활용합니다.

## 사용 팁

- "오늘 아침 브리핑 해줘"처럼 물으면 매출·방문·폐기·재배치 기회를 요약해 줍니다.
- 답변이 유용하면 Task로 등록해 실행 항목으로 이어가세요.
