## 스토어에이전트가 하는 일

스토어에이전트는 매장 데이터를 대신 읽고, 다음 한 수를 제안하는 **대화형 운영 코파일럿**입니다. 매출·방문·재고·직원·CCTV 데이터를 한자리에서 묻고, 답을 받아 바로 실행으로 옮깁니다.

**물어보면 화면이 열립니다**

"오늘 아침 브리핑 해줘", "도시락 발주 추천해줘"처럼 자연어로 물으면, 그에 맞는 분석 화면이 열립니다.

**근거까지 되짚습니다**

추천에는 항상 근거가 붙습니다. "왜 이 수량인지"를 표와 데이터로 확인하고 판단하세요.

**결정은 사람이 합니다**

권고는 AI가, 결정은 사장님이 합니다. 마음에 드는 답변은 Task나 메모로 등록해 실행으로 이어갑니다.

## 이 매뉴얼 구성

- **시작하기** — 로그인, 매장 선택, 오버뷰 화면, AI 채팅 사용법
- **분석 활용** — 차트 질문, 이상 감지, 발주·선반·직원 분석, 외부 컨텍스트
- **제품 매뉴얼** — Task·메모 등록, 응답 중단, 자주 묻는 질문
