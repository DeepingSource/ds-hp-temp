陳列棚分析は、店舗内の各陳列棚の成果を詳細データで測定する機能です。陳列棚全体はもちろん棚(段)単位で顧客の訪問、露出、探索、ピックアップ行動を数値化し、表形式で提供することで、陳列戦略の改善優先順位を決定できる主要ツールです。

## 主要機能

### 陳列棚ファネル分析

- 顧客の行動を 訪問 → 露出 → 探索 → ピックアップ の4段階ファネルで測定します。
- 各段階別のコンバージョン率を数値で確認し、どこで顧客の関心が途切れるか(ボトルネック区間)を精密に診断します。

**活用例**

- 多く見られたがピックアップが少ない陳列棚や棚を識別し、商品の魅力度・価格・陳列方式の問題点を推定
- 露出 → 探索のコンバージョン率が低い陳列棚を見つけ、商品情報提供方式(POP)を改善し効果検証

### 陳列棚および棚の詳細表示

- 陳列棚全体単位だけでなく、棚(段)単位でも成果データを詳細に確認します。
- 性別/年齢層フィルターを適用し、ターゲット顧客層が特定の陳列棚や棚にどう反応するかを区別して分析します。

**活用例**

- 上段、中段、下段の棚別ピックアップ率を比較し、「ゴールデンゾーン(Golden Zone)」を定義して商品配置を最適化
- 特定グループが特定の陳列棚の「中段」棚でのみピックアップするパターンを把握し、ターゲット向けの陳列戦略を立案

## 主要指標 (KPI)

| 指標名 | 説明 | 活用インサイト |
| ------ | ---- | ------------- |
| **訪問→露出比率** | 通過した顧客のうち実際に見た比率 | 陳列棚の露出力・注目度の診断 |
| **露出→探索比率** | 見た顧客のうち一定時間以上関心を持って見た比率 | 陳列商品の魅力度および情報伝達力の評価 |
| **探索→ピックアップ比率** | 探索した顧客のうち実際にピックアップした比率 | 関心が購買行動につながる程度の測定 |
| **棚(段)別成果** | 各棚単位の露出、ピックアップ比率 | 商品位置別の差異分析、配置戦略改善の根拠 |

## 活用事例

### 商品配置と陳列の最適化

- 露出は高くピックアップが低い陳列棚の問題点を診断し、価格競争力・VMD・照明などの改善ポイントを抽出
- 棚単位の成果データを活用し、商品の最も効率的な高さを見つけて配置戦略を精緻化

### プロモーション成果の定量検証

- プロモーション前後の特定陳列棚の数値変化を確認し、広告やイベントの実際の効果を客観的に測定
- 成果が低い陳列棚を識別し、改善優先順位を決定してリソースの浪費を防止

### 運営効率と在庫戦略の強化

- 売上貢献度が高いか、ポテンシャルの大きい陳列棚や棚に在庫と人員を集中配置し、運営効率を最大化
- 顧客の視線が集中する「露出」段階と実際の行動である「ピックアップ」段階を分離分析し、陳列戦略の効果を判断

## 上級活用ヒント

> **💡 팁**
>
> **戦略実行前後の比較分析** — イベント前後の陳列棚の露出 → ピックアップのコンバージョン率変化を比較し実際の効果を確認、同じ商品を異なる陳列棚位置(例: 入口 vs 奥)に配置してピックアップコンバージョン率を比較測定します(A/B テスト)。

> **💡 팁**
>
> **ターゲット顧客群別の最適化** — 性別・年齢層別に陳列棚成果をフィルタリングして確認し、ターゲット顧客に最も反応の良い陳列棚と棚にマーケティングリソースを集中配置します。

> **💡 팁**
>
> **棚効率の標準化** — 上段・中段・下段の棚別の露出およびピックアップ率パターンを分析し、最も成果の良い棚パターンを抽出し、最適な棚パターンを他の陳列棚にも拡大適用して、店舗全体の陳列戦略の標準を確立します。

## まとめ

> 陳列棚分析は、店舗内の各陳列棚と棚(段)の成果を、訪問、露出、探索、ピックアップの多段階ファネルで測定する主要分析ツールです。
> データに基づくコンバージョン率を通じて、商品配置、プロモーション、運営戦略を棚単位まで精緻に改善できる具体的な実行案を導き出します。
