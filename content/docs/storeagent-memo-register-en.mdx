For an answer that isn't something to act on right away but is worth remembering, you can save it as a memo. Where "Save as Task" manages things you need to do, "Save as Memo" keeps content you want to refer back to.

## Key features

- **Save as Memo button** — The button below an AI answer saves the answer as a memo.
- **Manager memo list** — Saved memos accumulate in the memo list along with their dates.
- **Add a memo directly** — Beyond AI answers, you can jot down whatever you need directly in the memo field.

## How to use

1. Find the AI answer you want to keep.
2. Tap "Save as Memo" below the answer.
3. Review the saved memo in the manager memo list.
4. If needed, add content directly in the memo field.

## Tips

A memo is a space to collect things freely, regardless of whether you'll act on them. Using tasks for things you'll actually need to do and memos for simple reference makes it easy to keep the two features distinct.
