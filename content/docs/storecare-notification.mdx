**알림함**은 매장에서 발생한 알림을 시간 순서대로 모아 보는 화면입니다. 하단 네비게이션의 종 아이콘에는 안 읽은 알림 수가 배지로 표시됩니다.

![알림함 목록](/images/docs/storecare/notification/_images/list.png)

*매장 필터 · 유형 필터 · 알림 목록*

## 상단 영역

- **매장 필터** — 왼쪽 위 `매장 전체 ⌄`에서 특정 매장의 알림만 골라 볼 수 있습니다(다매장 운영 시).
- **설정(⚙️) · 더보기(⋮)** — 알림 관련 설정과 추가 동작을 제공합니다.
- **전체 / 안읽음** — 전체 알림과 안 읽은 알림(개수 표시)을 전환합니다.

## 유형 필터

목록 위의 칩으로 보고 싶은 알림 유형만 추려 볼 수 있습니다.

| 유형 | 예시 |
| --- | --- |
| **진열 미흡** | 진열대 결품·나쁨 상태 (예: `냉장매대_간편식3 나쁨 누적 3회 감지`) |
| **청결 미흡** | 테이블·구역 청결 상태 |
| **운영 이상** | 계산대 자리 비움 등 운영 상황 |
| **온도 이상** | 냉장·냉동 설비 온도, `온도 정보 끊김`, `배터리 부족` 등 설비 알림 |
| **매장 요약** | 하루 매장 점수 요약 등 |

## 알림 목록

각 항목에는 **유형**, **매장 이름**, **발생 시각**, **메시지**가 표시됩니다. 같은 항목에서 알림이 반복되면 **모두 보기**로 묶이며, 누르면 발생한 시각들을 한 번에 볼 수 있습니다.

알림을 누르면 그 알림과 연결된 **상세 화면으로 바로 이동**합니다. 예를 들어 온도 알림을 누르면 해당 설비의 온도 상세가 열립니다.

![알림에서 이동한 온도 상세](/images/docs/storecare/notification/_images/detail.png)

*온도 알림을 누르면 해당 설비의 온도 상세로 이동합니다*

> **📌 참고**
>
> `온도 정보 끊김`·`배터리 부족`은 카메라·온도계 장비 쪽 신호입니다. 반복되면 해당 장비의 와이파이 연결이나 배터리를 점검해 주세요.
