매장 오버뷰는 화면 오른쪽에 매장 지표를 상시 표시하는 대시보드입니다. 고객 분석 4개, 매장 분석 4개 지표를 한눈에 확인할 수 있고, 상단 기간 필터로 조회 기준을 바꿀 수 있습니다. 아래로는 신제품·뉴스와 추천 액션·점장 메모가 이어집니다.

## 주요 기능

- **고객 분석 4종** — 일별 방문자 수, 남녀 방문 비율, 연령별 방문 비율, 전환율을 제공합니다.
- **매장 분석 4종** — 일별 품절 추세, 일별 청소 점수, 장바구니 분석, 매대 전환율을 제공합니다.
- **기간 필터** — 평일·주말·최근 2주·전월 기준으로 지표를 전환합니다.
- **연령대 구분** — 연령별 방문 비율은 30대 미만·30대·40대·50대·60대 이상으로 나뉩니다.

## 사용 방법

1. 상단 기간 필터에서 평일·주말·최근 2주·전월 중 하나를 선택합니다.
2. 고객 분석 영역에서 방문자 수·성별·연령·전환율 흐름을 확인합니다.
3. 매장 분석 영역에서 품절·청소·장바구니·매대 전환율을 점검합니다.
4. 관심 있는 지표가 보이면 왼쪽 채팅으로 관련 질문을 이어갑니다.

## 사용 팁

- 같은 지표라도 평일과 주말 필터를 번갈아 보면 운영 패턴 차이가 드러납니다.
- 장바구니 분석의 배수 값이 높을수록 함께 팔리는 조합이 강하다는 의미입니다.
