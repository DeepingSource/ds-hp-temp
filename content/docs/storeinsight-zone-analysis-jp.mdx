- 店舗を複数のゾーンに分け、各ゾーンの訪問者数や滞在時間などを分析します。
- どこに顧客が多く滞在し、関心を持っているかを把握できます。
- ホットゾーン / コールドゾーンの区分、商品の再配置、ターゲットマーケティングに有用です。



ゾーン別分析は、店舗を「ゾーン(Zone)」単位に分けて、顧客がどの空間でどのくらい滞在し、どのような関心を示したかを分析する機能です。ゾーン別の流入、滞在、関心度、人口統計情報をもとに、**店舗レイアウトと陳列戦略をデータドリブンで最適化**できます。

ゾーンの定義は[ゾーン設定](/jp/resources/docs/storeinsight-zones)ガイドをご参照ください。

### ゾーン設定

- 店舗フロアプラン上に自由にゾーンを指定できます(多角形、円形など)。
- 商品種類、機能、ブランドなど、さまざまな基準で分けることができます。

**活用例**

- 新商品ゾーン、イベントゾーン、カウンター待ちゾーンなどに分けて空間別の成果を比較
- 特定ブランドゾーンの顧客反応分析およびプロモーション効果の測定

### 訪問者数 & 滞在時間分析

- 各ゾーンに入った訪問者数(流入量)と平均滞在時間を自動的に集計します。
- 滞在時間フィルター(例:3秒以上)を活用して、特定時間以上滞在した顧客のデータを確認できます。

**活用例**

- 訪問者数と平均滞在時間の比較によるゾーンの実質的な魅力度の判断
- 滞在時間が低いゾーンの陳列状態の改善および変化の測定
- 訪問者数データに基づく人員配置およびリソース投入の妥当性の評価

### ゾーン別人口統計分析

- 各ゾーンを訪れた顧客の**性別および年齢**分布を自動的に分析します。
- 顧客属性(性別、年齢層)を基準にデータをフィルタリングして照会できます。

**活用例**

- 顧客属性(性別、年齢層)を基準にデータをフィルタリングして照会できます。
- **顧客群別の滞在時間分析**による優先改善またはマーケティング集中対象の選定
- **ターゲット顧客向けカスタマイズ広告**および店舗内サイネージ運営戦略の立案

## 主要指標サマリー (KPI)

| 指標名 | 説明 | インサイト |
| --- | --- | --- |
| ゾーン別訪問者数 | 該当ゾーンに入った顧客数 | ゾーンの人気(集客力)の確認 |
| 平均滞在時間 | 顧客がそのゾーンに滞在した平均時間 | 関心度および空間魅力度の評価 |
| 訪問密集度 | 顧客が集まっている度合い | ホット/コールドゾーンを一目で把握(ヒートマップ可視化) |
| ゾーン別人口統計 | 性別および年齢分布 | 顧客群別の好みの空間の分析 |
| ゾーン訪問率 | 全顧客のうち、当該ゾーンを訪れた比率 | アクセシビリティおよびゾーン到達率の診断 |
| バウンス率 | ゾーンに入ってすぐに離脱した比率 | ゾーンの第一印象および商品適合度の診断 |

## 期待される効果

### 空間最適化

- 顧客動線と滞在データに基づく効率的な店舗レイアウト設計が可能
- ホットゾーンは維持/強化、コールドゾーンは改善する戦略の立案

### ターゲットマーケティング

- 顧客層別の好みのゾーンの確認による、的確なターゲットプロモーションの実行
- ゾーン別の広告・陳列コンテンツの差別化

### 運営改善

- 混雑時間帯およびゾーンの把握による、人員・在庫配置の最適化
- ボトルネック/離脱ゾーンの改善によるショッピング利便性の向上

## 活用事例

- **ホットゾーン / コールドゾーン診断** — どのゾーンが人を引き付け、どこが空いているかを一目で把握
- **商品再配置** — コールドゾーンに置かれた商品の位置・露出方法の再検討
- **ターゲットマーケティング** — 人気ゾーンの訪問者プロフィールに基づいたキャンペーンメッセージの設計

## 上級活用ヒント

### コアゾーンの集中分析

- 売上貢献度の高いゾーンの顧客特性、滞在行動を精密に分析
- 関連商品の陳列、照明、コンテンツ強化による追加成果の創出

### 不人気ゾーンの改善

- コールドゾーンの原因(位置、構成、陳列など)を把握した後、改善案を適用してデータ検証
- VMD、陳列商品、プロモーション変更後の流入および滞在時間の変化の測定

### シーズン/イベント比較

- シーズンごとの顧客反応の差を比較し、時期別の戦略を立案
- お盆、イベントなど前後比較による空間および商品配置の最適化

> **💡 팁: まとめ**
>
> ゾーン別分析は、店舗を「顧客が実際に反応した空間単位」に分けて成果を測定するツールです。単に空間を分けるだけでなく、顧客の行動(滞在/関心)を基準として各空間の効率をデータで把握し、効率的な改善戦略の立案を支援します。
