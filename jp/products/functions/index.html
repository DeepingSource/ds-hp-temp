機能saai care検知 · 今saai insight分析 · 昨日saai agent提案・運営 · 次
count来店・滞在人数今の滞在人数・混雑度を検知し、閾値超過を通知日・週・月の来店時系列、曜日・時間帯・前年比の推移来店予測に基づく人員配置・営業時間・販促タイミングの提案
census匿名の人口統計今の来店客構成(年齢層・男女比の推定)を把握時間帯・曜日ごとの来店客セグメントの変化を分析ターゲット層に合わせた商品構成・時間帯運営の提案
trail動線特定エリアの異常な滞留・不自然な動線を検知エリア別の通過・滞在ヒートマップ、主要経路の分析陳列・レイアウト・案内動線の改善提案
gaze視線 · gaze→pick新規陳列・POP前の視線反応をリアルタイムに確認棚・商品別の注目度(gaze→pick 転換)を分析ゴールデンゾーン配置・陳列優先度の提案
wait待ち・行列レジ・待機列の閾値超過をリアルタイム通知待ち時間のピーク、曜日・時間帯の分析人員再配置・レジ増設・セルフ誘導の提案
tide混雑(満ち引き)店舗全体の混雑の満ち引きをリアルタイム検知混雑の周期・ピークパターンを分析混雑緩和の運営(人員・販促の分散)提案
keepロス防止異常行動・離脱・紛失リスクをリアルタイム検知ロス発生のエリア・時間・パターンを分析什器配置・動線遮断・重点管理エリアの提案
shelf棚・在庫の状態欠品・乱れ・価格表示の誤りをリアルタイム検知欠品頻度・回転率・棚別の成果を分析補充サイクル・フェイシング・陳列優先度の提案
motion行動異常行動(転倒・トラブル等)をリアルタイム検知店内の行動タイプ・頻度を分析安全・サービス介入ポイントの提案
fitトレンド適合新商品の初期反応の異常を検知商品・店舗のトレンド適合度を分析発注・商品構成・バイヤーの意思決定を支援
pop販促物(POP)掲出中のPOPの露出・破損状態を検知POP別の注目・転換効果を分析POP文言・位置・交換サイクルの提案・生成
talk多言語接客外国語・ヘルプ要請の状況をリアルタイム検知問い合わせの種類・言語・頻度を分析多言語接客・案内文言の提案・生成

count は機能です — 4番目の製品ではありません。

store count

読み方

「一つの count が、care ではリアルタイム検知に、insight では推移分析に、agent では運営提案になる」。一つのマスはこう読みます — saai care が store count を回す。

拡張は一行

新しい機能は一行を足すだけ。3つのモードの体験が自動で生まれ、新しい製品も新しい名前も要りません。
