### 4. Spatial AI · MTMC 섹션 모션 발전
- 파일: `src/components/mockups/SpatialTrajectoryMockup.tsx` (framer-motion + `useMockupLoop`, PATH_POINTS 9개 하드코딩)
- 어색한 지점 진단 → 개선 방향:
  - 포인트 이동이 900ms 등속 step이라 "순간이동" 느낌 → 경로를 SVG path 기반 보간(`offsetDistance` 또는 framer-motion `animate` along path)으로 연속 이동

  - 라벨 파이프라인 1600ms 루프와 이동 루프의 주기가 어긋남 → 하나의 타임라인으로 동기화
  - easing: linear → `easeInOut`, 시작/정지 시 감속
- 산출물: 개선안 프로토타입 1개 → 검토 후 반영. `usePrefersReducedMotion` 폴백 유지
